Welkom bij Sergei's Programmeerweetjeshoek

Elke week bellen wanhopige luisteraars, van welke het merendeel knappe dames onze service, met wanhoop. Vandaag leest u het relaas van een programmeuse uit een pittoresk dorp in het Antwerpse.

Q: Hallo! Ik zou graag Notepad++ of Atoms openen via m'n git bash, maar ik voel me als een brei-oma zonder naalden.

Ha, wij houden hier op de redactie wel van brei-oma's! Dankjewel voor deze boeiende vraag, om in de sfeer te blijven zal ik de rode draad proberen bloot te leggen.

Ok, laten we beginnen bij het begin. Ik ga er hier vanuit dat je de git shell gebruikt van deze pagina.

Concreet begrijp ik uit je vraag dat je graag in de command line een commando wil intypen als "notepad++ dezeTekstWilIkOpenen.txt", hiervoor neem je volgende stappen:

  • windowstoets + S --> zoek naar 'git shell'
  • navigeer naar een plaats naar keuze en maak een bestandje aan, bv. 'cd c:/users/sergei/desktop', vervolgens 'touch verhaaltje.txt'
  • Controleer of je bestandje echt aangemaakt is door het commando 'ls' te typen, je bestand moet tussen de opgelijste bestanden staan
  • windowstoets + S --> zoek naar 'Rapid Environment Editor', open nu als administrator dit programma
  • Sluit de terminal af
  • In rapid environment editor in de linkse kolom, druk je op het '+'-tekentje naast 'Path'
  • Rechtermuisklik op 'Path' en klik op 'add value', vervolgens op 'insert directory path'
  • Nu moet je op zoek naar het '.exe'-bestandje van het gewenste programma
  • Bij mij zit dat voor Notepad++ bij 'Deze pc' - 'Windows8_OS (C:)' - 'Program Files (x86)' - 'Notepad++'
  • Ik klip op het 'Notepad++'-mapje en klik op de 'ok'-toets
  • Druk nu linksboven in 'Rapid Environment Editor' op het diskette-symbool en sluit het programma
  • Nu weer stap 1 -> open git shell
  • Navigeer naar de plaats waar ons verhaaltje.txt is opgeslaan. Typ vervolgens 'notepad++ verhaaltje.txt'
  • Notepad++ opent verhaaltje.txt
